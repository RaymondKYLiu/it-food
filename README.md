## 推薦菜色

- 海鮮 - 淡菜(mussel) 蛤蜊(clam) 長腳蝦 非常鮮甜
- 義大利麵 - 通常清炒海鮮口味的都不會雷 墨魚麵很鹹 蛋黃培根麵有點膩
- 前菜 - 火腿一定要試一下 可搭配 Burrata Cheese + 芝麻葉, 哈密瓜
